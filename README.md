# 個人房貸試算器

## 專案簡介
本專案為「視窗程式設計（II）」作業一，使用 C# Windows Forms 製作一套個人房貸試算器。  
使用者可輸入房屋總價、自備款比例或自備款金額、年利率、貸款年限以及寬限期，程式會自動計算貸款總金額、每月應繳金額、首期利息、首期本金、總利息支出與總還款金額。

## 開發環境
- Visual Studio
- C#
- Windows Forms

## 功能說明
本程式具備以下功能：
1. 輸入房屋總價
2. 輸入自備款比例或自備款金額（二擇一）
3. 輸入年利率
4. 輸入貸款年限
5. 輸入寬限期
6. 計算貸款總金額
7. 顯示每月應繳金額
8. 顯示首期利息與首期本金
9. 顯示總利息支出與總還款金額
10. 提供清除功能，方便重新輸入資料

## 操作方式
1. 開啟程式後，於左側輸入房貸相關資料。
2. 房屋總價請輸入房屋價格。
3. 自備款可選擇輸入「比例」或「金額」，請擇一輸入。
4. 輸入年利率、貸款年限與寬限期。
5. 按下「開始試算」按鈕。
6. 程式會在右側顯示試算結果。
7. 若要重新輸入資料，可按下「清除」按鈕。

## 計算公式
### 1. 貸款總金額
- 使用自備款比例時：  
  貸款總金額 = 房屋總價 × (1 - 自備款比例)

- 使用自備款金額時：  
  貸款總金額 = 房屋總價 - 自備款金額

### 2. 月利率
月利率 = 年利率 ÷ 12 ÷ 100

### 3. 本息平均攤還公式
每月應繳金額 M = P × [r(1+r)^n] ÷ [(1+r)^n - 1]

其中：
- P：貸款本金
- r：月利率
- n：總期數

### 4. 寬限期
若設定寬限期，則寬限期內每月僅繳利息，不償還本金。  
寬限期結束後，再依剩餘期數重新計算每月本息攤還金額。

## 輸入驗證
本程式包含基本輸入驗證：
- 房屋總價需大於 0
- 年利率不可小於 0
- 貸款年限需大於 0
- 寬限期需大於等於 0，且不可大於等於貸款年限
- 自備款比例需介於 0 到 100 之間
- 自備款金額不可小於 0，且不可大於等於房屋總價
- 自備款比例與自備款金額不可同時輸入
- 若輸入不合法，會跳出錯誤訊息提醒使用者
